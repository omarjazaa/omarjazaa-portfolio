# Omar Al-Jazaa Portfolio

## Features
- Fully responsive portfolio with cybersecurity theme
- English/Arabic i18n with RTL support
- Dark/light theme toggle
- Smooth animations and scroll reveal
- Contact info, projects, experience showcase

## Tech Stack
- React 18 + Vite + TypeScript
- Tailwind CSS + shadcn/ui components
- Lucide React icons
- react-router-dom, react-query

## Development
```bash
npm install
npm run dev
```

## Deployment
- Netlify/Vercel: Connect GitHub repo
- All assets in `/public`, static build ready
